# vagrant-mt-dev
Vagrantfile for running/developing MT

* Can run MT
* Can execute MT tests
* Can build MT
* Can build Riot.js files
* Can build a-table.js

## Requirements
* VirtualBox
* Vagrant
  * vagrant-hostmanager

## Environments
* Ubuntu 12.04.5 LTS
* Apache HTTP Server 2.2.22
* MySQL 5.5.54
* Memcached 1.4.13
* Perl 5.14.2
  * App::cpm
  * Cache::File
  * Image::Magick 6.6.9
  * CPAN modules in cpanfile of MT develop branch
* PHP 5.3.10
  * mysql module
  * mysqli module
  * pdo_mysql module
  * memcache module
  * gd module
  * PHPUnit 4.8.36
  * phpMyAdmin 3.4.10.1deb1
* PhantomJS 1.9.8
* Node.js 4.8.3
  * Yarn
* git 1.7.9.5
